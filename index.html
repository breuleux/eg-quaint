What is it?

Quaint is a markup language that you can use to write documents. It is similar to Markdown, but it is more powerful and more extensible.

Install

To install Quaint globally:

npm install quaint -g
This will install the quaint command on your system. Then you can convert quaint markup to HTML! See usage.
